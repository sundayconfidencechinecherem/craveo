# Craveo ~ The Food Social Media App

Craveo is a **food-focused social media platform** that allows users to share, discover, and engage with food-related content in real-time. This repository contains the complete app structure, including frontend, backend, and project documentation.


### Overview

Craveo allows users to:
- Browse and engage with food posts from the community
- Like, comment, and share posts
- Create and manage their own posts
- Discover trending dishes, recipes, and culinary experiences
- Follow other users and interact with a dynamic feed

The app focuses on **real-time user interaction** and a **responsive interface** across web, mobile, and PWA platforms.


### Technologies

**Frontend:**
- Next.js (React framework)
- TypeScript
- TailwindCSS
- PWA support

**Backend:**
- Node.js
- Express
- MongoDB
- GraphQL API

**Others:**
- API Integration for dynamic feed
- State management for interactive components


### Key Features

- Dynamic feed loading using GraphQL
- Like, comment, and share posts in real-time
- Infinite scrolling and pagination for seamless content browsing
- Responsive design for mobile, tablet, and desktop
- Smooth animations for enhanced user experience
